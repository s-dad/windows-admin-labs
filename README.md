# Windows Admin Labs

A collection of hands-on Windows Server administration labs completed
as part of ITEC 350 at Franklin University. Each unit covers a core
sysadmin skill set including AWS EC2 provisioning, Active Directory,
Group Policy, IIS, PowerShell automation, and AWS load balancing.

## Skills Demonstrated
- Windows Server 2019 configuration on AWS EC2
- Active Directory — users, groups, OUs, domain controllers
- DNS and Group Policy management
- IIS web server setup and automation
- PowerShell scripting for service and AD management
- AWS Application Load Balancer configuration
- Security auditing and vulnerability scanning with Nessus
